# 快速存档功能 · Abyss Quick Save

给本游戏添加「快速存档 / 快速读取」的 RPG Maker MV 插件。一个快捷键存档、一个快捷键读档，写入的是游戏自带读取界面里的 **第 20 号存档槽**，用官方界面即可读取。

> 制作时对应游戏版本 **1.1.3 rev.2**。无其他版本完整测试，不保证在旧/新版本中正常运行。**安装前请自行备份 `www/save` 与 `www/js/plugins.js`。**

---

## ✨ 功能一览

| 快捷键 | 作用 |
| --- | --- |
| `Ctrl + Shift + S` | 保存到界面显示的第 20 号槽 |
| `Ctrl + Shift + L` | 打开官方读取界面并定位到第 20 号槽 |
| `Ctrl + Shift + Delete` | 删除快速存档并停用功能 |

主菜单里还会多出一项 **「快速存档管理」**，可手动保存 / 读取 / 停用 / 卸载。

---

## 🛡️ 保存安全限制

为避免存档异常或破坏游戏流程，以下情况 **不会写入存档**（但仍可正常**快速读取**）：

- 战斗中
- 敌人警戒或追击中
- 附近存在敌人
- 脱战不足约 1 秒
- 在家（剧情区域）
- 剧情 / 事件演出过程中

此外：每次保存都会弹出第 20 号槽 **覆盖确认**；地面有掉落物时会额外提示。

## 📦 安装

### 方式一：自动安装（推荐）

1. 把压缩包解压到 **游戏根目录**（与 `Game.exe` 同一文件夹），出现覆盖 / 合并提示时选「是」。
   解压后应能看到 `www\js\plugins\Abyss_QuickSave.js`。
2. 双击运行 **`安装快速存档功能.cmd`**（只需运行一次）。
   它会自动把插件登记进 `www\js\plugins.js`，并生成备份 `plugins.js.before_Abyss_QuickSave_install`。
3. 启动游戏，按 `Ctrl + Shift + S` 试一下即可。

> ⚠️ 插件必须登记进 `plugins.js` 才会生效，**只复制 `.js` 文件是不够的**。

### 方式二：手动安装（自己改 plugins.js）

1. 确认插件文件已就位：`www\js\plugins\Abyss_QuickSave.js`。
2. 用记事本打开 `www\js\plugins.js`，内容形如：

   ```js
   var $plugins =
   [
   {"name":"semver", ... },
   ...
   {"name":"2den1580_PatreonVerify", ... }
   ];
   ```

3. 在最后一个条目末尾加一个英文逗号 `,`，另起一行，在 `];` 之前粘贴这一整行：

   ```js
   {"name":"Abyss_QuickSave","status":true,"description":"Abyss quick save with safe removal","parameters":{}}
   ```

   改完后结尾应是（注意倒数第二行末尾的逗号）：

   ```js
   {"name":"2den1580_PatreonVerify", ... },
   {"name":"Abyss_QuickSave","status":true,"description":"Abyss quick save with safe removal","parameters":{}}
   ];
   ```

4. 保存 `plugins.js`（编码保持 **UTF-8**）。务必保证：
   - 这一行加在 `];` 的前面；
   - 与上一行之间用英文逗号隔开；
   - 它自己末尾不要加逗号（因为它是最后一个）。
5. 启动游戏即可。

> 建议改之前先把 `plugins.js` 复制一份做备份，万一改错可以还原。

---

## 🧹 卸载

- **方式 A（脚本）**：双击运行 `卸载快速存档功能.cmd`。它会从 `plugins.js` 移除登记行并删除插件文件；**不会删除你的任何存档**。
- **方式 B（游戏内）**：主菜单 →「快速存档管理」→「彻底卸载快速存档功能」。
- **方式 C（手动）**：把安装时加进去的那一行删掉，并删除 `www\js\plugins\Abyss_QuickSave.js`。

---

## 📝 说明

- 快速存档写入的是官方第 20 号槽（内部 `file21.rpgsave`），用官方读取界面即可读取。
- 安装 / 卸载脚本只改 `plugins.js` 和插件文件，**不会动你的存档文件**。
- 如遇问题，可还原 `plugins.js` 的备份文件后重试。
